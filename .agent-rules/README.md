# Reglas para Agentes de IDE

Esta carpeta contiene reglas y guías para agentes de IA que trabajan en este proyecto.

## Estructura

- `project-context.md` - Contexto general del proyecto y arquitectura
- `coding-standards.md` - Estándares de código y convenciones
- `api-guidelines.md` - Guías para desarrollo de APIs
- `frontend-guidelines.md` - Guías específicas para frontend
- `backend-guidelines.md` - Guías específicas para backend
- `database-guidelines.md` - Guías para base de datos y Prisma
- `testing-guidelines.md` - Guías para testing
- `git-workflow.md` - Flujo de trabajo con Git

## Uso

Los agentes deben consultar estos archivos antes de realizar cambios en el código para asegurar consistencia y calidad.

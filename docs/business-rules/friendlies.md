# Reglas de Negocio — Amistosos

## RN-022 — Seña para confirmar amistoso

- **Contexto**: Partidos amistosos.
- **Regla**: Una vez que la organización genera el partido a partir de una solicitud, **cada equipo debe pagar una seña** para confirmar el encuentro.

## RN-023 — Ventana de confirmación de 24 hs

- **Contexto**: Partidos amistosos.
- **Regla**: La confirmación del encuentro (seña) debe realizarse dentro de las **24 horas** posteriores a la generación del partido por parte de la organización.
